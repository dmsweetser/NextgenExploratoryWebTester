Known Bugs and Edge Cases for Testing
  • Validation Errors: Click "Show Validation Errors" to reveal various error messages in red.
  • Unusual System Errors: The unusual error message appears in a monospace font with a different background.
  • Low Contrast Errors: Some errors have low contrast that might be hard to detect.
  • Hidden Field Errors: Click "Toggle Hidden Field" to show/hide a field with its associated error.
  • Overlay Errors: Click "Show Overlay" to display an overlay with potential error messages.
  • Dynamic Overlay Errors: Click "Show Dynamic Error" to display an error followed by an overlay.
  • Delayed Errors: Click "Show Delayed Error" to see an error that appears after a delay.
  • Conditional Errors: Some errors only appear based on specific conditions (like selecting certain options).
  • Hover Errors: Some errors only appear when hovering over elements.
  • Broken Link: The "Broken Link (404)" in the dropdown menu leads nowhere.
  • Error Link: The "Error Link" in the dropdown shows an error message when clicked.
  • Invisible/Zero-sized Elements: These should not be visible or interactable.
  • Dynamic Select: Selecting "Option 3" reveals a dependent field with its own error.
  • Conditional Select: Selecting anything other than "Option B" shows an error.
  • Random Errors: Some actions have a random chance of showing errors.
  • System-Level Errors: The unusual error simulates a system-level failure, not just validation.
