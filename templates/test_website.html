Known Bugs and Edge Cases
  • Validation Errors: Click "Show Validation Errors" to reveal hidden error messages.
  • Hidden Field: Click "Toggle Hidden Field" to show/hide a field dynamically.
  • Broken Link: The "Broken Link (404)" in the dropdown menu leads nowhere.
  • Hover Content: The "Hover over me" div reveals content only on hover (should not be visible to the bot unless it simulates hover).
  • Invisible/Zero-sized Elements: These should not be visible or interactable.
  • Dynamic Select: Selecting "Option 3" reveals a dependent field. The "Update Select Options" button changes the select options dynamically.
  • Simulated Error: Click "Simulate Error" to show an error message.
  • Slow Response: Click "Slow Response (3s)" to simulate a slow API call.
