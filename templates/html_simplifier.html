What the Simplifier Does

The HTML simplifier removes non-essential elements and attributes to help NEWT focus on the semantic content of the page. This includes:

  • Removing scripts, styles, and other non-visible elements
  • Removing hidden elements (display:none, visibility:hidden)
  • Removing non-semantic attributes
  • Preserving only visible, meaningful content
  • Keeping semantic attributes like id, class, name, href, etc.

This helps NEWT focus on functionality rather than presentation, making it more effective at finding real bugs.
